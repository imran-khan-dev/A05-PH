Question-1: What are the different ways to select an element in DOM?

To select an element in the DOM, we can use a few methods. getElementById() picks an element by its ID, getElementsByClassName() picks by class name, and getElementsByTagName() picks by tag name. If more control is needed, querySelector() gets the first match and querySelectorAll() gets all matches. Also, closest() helps find the nearest element that matches a selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference between innerHTML, innerText, and textContent is that innerHTML gets or sets the HTML content inside an element, including any tags. innerText gets or sets the visible text, ignoring HTML tags and considering styles like hidden elements. textContent also gets or sets the text, but it doesnâ€™t care about styles and includes all text, even hidden ones. So, innerHTML gives the full HTML, innerText gives visible text, and textContent gives all text, ignoring styling.

Question-3: What is the event Delegation in DOM?

Event delegation in the DOM is a technique where a single event listener is added to a parent element instead of multiple child elements. When an event happens on a child, it bubbles up to the parent, which then handles it. This is useful for handling events efficiently, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is when an event starts from the target element and moves up to its parent elements. If a child element is clicked, the event first runs on that element, then bubbles up to its parent, then to the grandparent, and so on, until it reaches the root. This allows parent elements to handle events triggered by their children.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements in JavaScript, we use document.createElement() to create a new element, then use appendChild() or append() to add it to the DOM. For example, we can create a new

and add it like this: let div = document.createElement('div'); document.body.appendChild(div);. To remove an element, we can use removeChild() or remove(), like div.remove();, to remove it from the DOM. This lets us dynamically manage elements on the page.
